# 键盘快捷键

## 编辑器

* Ctrl+\：自动补全
* Ctrl+B：跳转到声明
* Ctrl+I：查看表达式信息
* Alt+F7：查找使用
* Alt+Shift+F：格式化
* Ctrl+G：跳转到行
* Ctrl+E：删除当前行
* Ctrl+S：保存当前编辑器文件
* Ctrl+Q：关闭当前编辑器
* F11：编辑器全屏
* Shift+Tab：自动缩进
* Ctrl+]：缩进
* Ctrl+[：反缩进

## 焦点

* Ctrl+D：切换编辑器/窗口组 tab
* Ctrl+0：焦点切换到编辑器
* Ctrl+1：焦点切换到文件树
* Ctrl+4：焦点切换到输出窗口
* Ctrl+5：焦点切换到搜索窗口
* Ctrl+6：焦点切换到通知窗口

## 运行

* F6：构建 & 运行
